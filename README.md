📊 Overview
A collection of interactive dashboards built with Microsoft Power BI, showcasing data modeling, DAX measures, and visualization best practices.

🥘 Zomato Restaurants Analysis
This dashboard analyzes Zomato restaurant data across 15 countries and 141 cities. It highlights:
1. Restaurant counts by city and cuisine
2. Ratings, votes, and customer engagement
3. Table booking vs. online delivery trends
4. Cuisine popularity (North Indian, Chinese, Fast Food, etc.)

⚙️ Tech Stack
1. Power BI Desktop (.pbix)
2. DAX for KPIs and measures
3. Structured datasets with country/time dimensions

🚀 Usage
1. Clone/download the repo.
2. Open .pbix files in Power BI Desktop.
3. Explore dashboards with filters and drill‑through features.
